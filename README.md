# Giant Wheel using opengl


![Alt Text](media/wheel.gif)

## Install Opengl

```
   sudo apt-get install g++
   sudo apt-get install freeglut3-dev
   
```

### Compile and run

```
g++ giant_wheel.cpp -lGL -lGLU -lglut
./a.out

```
